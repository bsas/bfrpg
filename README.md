# Foundry Virtual Tabletop - Basic Fantasy Role-Playing Game

This game system for [Foundry Virtual Tabletop](http://foundryvtt.com) provides character sheet and game system support for the [Basic Fantasy Role-Playing Game](https://www.basicfantasy.org/) system.

This system provides character sheet support for Actors and Items, mechanical support for dice and rules necessary to play games of BFRPG, and compendium content for Monsters, Items, Spells and more!

## Installation Instructions

To install and use the BFRPG system for Foundry Virtual Tabletop, simply paste the following URL into the 
**Install System** dialog on the Setup menu of the application.

https://raw.githubusercontent.com/bsas/bfrpg/main/system.json

If you wish to manually install the system, you must clone or extract it into the ``Data/systems/bfrpg`` folder. You
may do this by cloning the repository or downloading a zip archive from the
[Releases Page](https://github.com/bsas/bfrpg/releases).
